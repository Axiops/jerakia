## Configuration

Jerakia by default reads from a configuration file called `/etc/jerakia/jerakia.yaml`

{% highlight none %}
$ mkdir /etc/jerakia
$ vim /etc/jerakia/jerakia.yaml
{% endhighlight %}

At a minimum, 

{% highlight none %}
$ puppet module install crayfishx/jerakia
{% endhighlight %}
